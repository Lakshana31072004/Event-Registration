# Event Registration System

A web-based **Event Registration System** built using **PHP**, **HTML**, **CSS**, and **MySQL**.  
This project allows users to register for events and provides an admin dashboard for managing registrations, sending emails/WhatsApp messages, exporting data, and more. :contentReference[oaicite:2]{index=2}

## 🧾 Features

✅ User registration form  
✅ Event registration handler  
✅ Admin login and dashboard  
✅ Email confirmation after registration  
✅ WhatsApp notification support  
✅ Export registered user data to Excel  
✅ Payment gateway integration (optional)  
✅ QR code generation  
✅ Configurable database support (MySQL)

## 📁 Technologies Used

- **PHP** – Backend logic  
- **HTML / CSS** – Frontend UI  
- **MySQL** – Database  
- **JavaScript** – Interactivity  
- **Bootstrap** – Responsive design  
- (Optional) **WhatsApp / Bulk WhatsApp integration**
